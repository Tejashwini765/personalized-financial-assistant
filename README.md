# Personalized Financial Assistant

This project helps users track and analyze expenses using machine learning.

## Features
- Expense tracking
- CSV upload
- Transaction classification
- Dashboard visualization
- Chatbot for financial insights

## Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI (Python)
- ML: Scikit-learn

## How to Run
1. Start backend:
   uvicorn main:app --reload

2. Start frontend:
   npm install
   npm run dev
